# ProtoTypeWar（試作機戦争）

軍工廠の管理者として、**部品の作成から試作機の設計まで、すべてを自由に作り上げ、試作機を量産して戦況を変える**ブラウザ・ストラテジーゲームです。

## 概要
あなたは軍工廠の管理者。素材を研究し、部品を作り、試作機（艦）を設計して軍に提出します。
戦略 AI が艦隊を編成・量産・運用し、戦線を押し上げて敵 HQ の制圧を目指します。
「工廠視点（設計に専念・AI が自動運用）」と「司令官（艦隊を手動指揮）」の 2 モードを切り替えながら戦況を動かしましょう。

## 遊び方（基本）
1. **研究**で素材をアンロックし、**部品作成**で性能を引き出す
2. 軍からの**依頼**を受け、要件を満たす**試作機を設計**して提出
3. 提出後の**改良**で量産艦を強化し、**戦域へ**で完成
4. 戦略 AI が量産・運用 → 戦線を押し上げて勝利

初回はタイトルからの**チュートリアル**で流れを案内します。詳しい操作・仕様はゲーム内**ヘルプ**を参照してください。

## 操作
- **マウス**：選択・移動指示・各種 UI 操作
- **WASD**：マップのスクロール
- 推奨環境：PC（キーボード＋マウス）／デスクトップ版 Chrome・Firefox（モバイル非対応）
